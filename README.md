# DoomsDay Fucker 

**Программа для обнаружения DoomsDay Client **

---

## 📌 Что это?

**DoomsDay Fucker** — это инструмент для Windows, который проверяет процессы `javaw.exe` на наличие признаков **DoomsDay Client** 

Программа использует два метода детекции: анализ памяти процесса и проверку через утилиту `jmap`.

---

## 🎯 Как это работает?

### Метод #1 — jmap анализ
- Запускает `jmap.exe` (утилита из JDK) для получения статистики классов
- Анализирует вывод на наличие аномальных символов и характерных паттернов, указывающих на вмешательство в класслоадер

### Метод #2 — Сканирование памяти
- Читает память процесса `javaw.exe`
- Ищет уникальные сигнатуры DoomsDay Client

### Метод #3 — Обнаружение процесса
- Находит все запущенные экземпляры `javaw.exe`
- Получает полные пути к исполняемым файлам
